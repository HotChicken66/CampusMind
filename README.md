# CampusMind - 全能型校园 Agent 接入框架

CampusMind 是一款专为**学生组织**深度定制的智能 QQ 机器人框架。它通过接入大语言模型（LLM）Agent，打通了知识录入、管理、评估与真实对话的闭环，旨在让每个部门都拥有一个专属的、懂业务、帮你干活的全能智能助手。

## 🌟 核心特色

- **应对封控的稳定连接**：推荐使用 **NapCat** 协议栈实现稳定接入，有效应对 QQ 平台的封控挑战，确保 Agent 长期在线，与同学们打成一片。
- **灵活的知识库架构**：
    - **双层架构**：创新设计了“周更知识”与“永久知识”双层体系。针对变动频繁的业务（如周会、动态通知），使用者可自由调整周更知识结构。
    - **QA 优先**：推荐使用 Q&A 对形式录入知识，能显著提升模型在复杂问询场景下的回答精准度。
- **全闭环管理与评估系统 (Visual UI)**：
    - **便捷管理**：提供可视化 UI 界面，支持知识的便捷导入与一键管理。
    - **深度评估打标**：内置评估 UI，针对真实对话记录进行打标。涵盖**专业知识、能力边界测试、提示词攻击防御、拟人闲聊**四大维度，实现数据的闭环调优。
- **拟人化社交互动**：
    - **融入校园文化**：Agent 具备发送表情包、斗图、发送 GIF 的能力（需自行配置资源），不仅是助手，更是群聊里的“氛围担当”。
    - **全能场景覆盖**：针对 QQ 群这一学生组织核心办公场景，集成了**智能闲聊、定时任务提醒、全方位知识问询**功能。

## 🏗️ 项目结构

- `bot.py`: 项目入口，初始化 NoneBot 核心。
- `src/plugins/`: 核心插件（RAG 工作流、打标拦截器、任务调度等）。
- `src/utils/`: 工具类（向量数据库、LLM 客户端、数据打标工具等）。
- `data/`: 用于存放知识库结构目录、表情包资源及本地缓存。
- `eval/`: 评估打标系统的前后端实现。

## 🚀 快速开始

### 1. 环境准备
确保您的 Python 版本 >= 3.10，并安装依赖：
```bash
pip install -r requirements.txt
```

### 2. 配置与启动
- 复制 `config.example.env` 并重命名为 `.env`，填入您的 API 密钥。
- 启动 **NapCat** 协议栈并配置上报。
- 执行 `start.bat` 一键开启机器人核心与管理后台。

### 3. 知识录入与打标
- 访问管理后台 UI，使用 QA 对形式导入部门知识。
- 运营一段时间后，在评估页面对真实日志进行打标，持续优化 Agent 表现。
